Globals

Headings

Wel doen

Ga uit van 1rem = 20px. We gebruiken een schaal van 1.333 per H-niveau.

Niet doen

Ga nooit bijvoorbeeld van <h1> naar <h3>, omdat dat er mooier uitziet. Gebruik headings in de correcte hiërarchische volgorde ( <h1> gevolgd door <h2>, enzovoort). Dit verbetert de leesbaarheid voor alle gebruikers. 

<header>
  <h1>Zo maak je een header</h1>
</header>

Gebruik voor hero <p class="hero">Zo maak je een hero</p>

Header{
  font-family: rigid-square, sans-serif;
}

h1 {
  font-size: 63.15px;
}

h2 {
  font-size: 47.37px;
}

  font-size: 35.54px;
}

  font-weight: 700;

  font-weight: 400;
}

  font-size: 15px;
  font-weight: 400;
}

header > h1 {

  text-align: right;

Kleuren

vtHBO gebruikt een breed kleurenpalet. Kleuren worden gebruikt om onderdelen te categoriseren. Binnen een kleur kunnen verschillende tinten worden gebruikt om hiërarchie te creëren.

Primaire kleuren

De primaire kleuren worden gebruikt voor essentiële elementen en tekst. De tinten kunnen gebruikt worden voor het creëren van hiërarchie

Donkerblauw

<p>
  color: #090940, #3a3a66, #6b6b8c, #9d9db3, #ceced9;
</p>

</p>

Secundaire kleuren

Deze kleuren kunnen vrij worden ingezet, bijvoorbeeld bij het kleurcoderen van categorieën of voor de verschillende stadia van interactieve elementen (hover, active etc.)

<p>
  color: #af1301, #fbc5b4, #169861, #63c09f, 
         #4fbbc2, #31439c, #8b3a00, #fe6f07;
</p>

Grijstinten

Gebruik grijs voor afscheidingen, elementen laag in de hiërarchie en voor inactieve elementen die geactiveerd kunnen worden

<p>
  color: #625c51, #c0beb9, #e0dedc, #f9f8f8;
</p>
